📱 DataMentor App
🔍 Overview

DataMentor is a mobile application designed to help users learn and interact with data concepts in a simple and engaging way. The app provides structured lessons, intuitive navigation, and a user-friendly interface to support learning.

🚀 Features
Interactive learning modules
Clean and intuitive user interface
Cross-platform support (Android, iOS)
Organized lesson structure
Smooth navigation and performance
🛠️ Technologies Used
Flutter (Dart)
Android Studio
Git & GitHub
