DataMentor 📊
🔍 Overview

DataMentor is a mobile application designed to help users learn data analytics concepts in a simple, interactive, and engaging way. It provides structured lessons and an intuitive interface for beginners and aspiring data analysts.

🚀 Features
Interactive learning modules
Clean and user-friendly interface
Structured lessons for easy understanding
Smooth navigation and performance
Cross-platform support (Android & iOS)
🛠️ Technologies Used
Flutter (Dart)
Firebase (optional if you used it)
Git & GitHub
📱 Platforms
Android (Available on Play Store)
iOS (Coming soon)
📚 Learning Topics
Excel for Data Analysis
SQL for Data Analysts
Python for Data Analysis
Power BI Dashboards
Tableau Visualization
